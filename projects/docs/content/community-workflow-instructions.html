Community Workflow Instruction's

In order to push your changes


1. Make sure you don't have any work in progress ( clean git workspace ). If you have some changes commit them or do git stash


2. Checkout to master and pull latest changes. git checkout master && git pull


3. Create new branch for yourself to work. git checkout -b my_new_branch_name


4. Make changes add them to staging area with git add command, then git commit and git push


5. Create a Merge Request ( you should see a link in console prompt to do this )
